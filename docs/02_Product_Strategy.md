# Product Strategy

## Product Vision

Forge is the operating system for turning ideas into reality. The platform
combines Team Formation, AI Guidance, Accountability, Project Execution, and
Product Launch into a single experience.

---

## Core Product Pillars

### Pillar 1 — Project Creation
Users need a place to store and develop ideas.
Features: Idea Vault, Project Creation, Validation, Roadmaps.

### Pillar 2 — Team Formation
Users need collaborators.
Features: Matching Engine, Team Recommendations, Builder Discovery, Compatibility Scoring.

### Pillar 3 — AI Guidance
Users need direction.
Features: AI Validation, AI Build Coach, AI Roadmaps, AI Recommendations.

### Pillar 4 — Accountability
Users need momentum.
Features: Milestones, Check-ins, Progress Tracking, Build Streaks.

### Pillar 5 — Launches
Users need outcomes.
Features: Launch Marketplace, Project Showcases, Product Discovery.

---

## Target Audience

**Primary:** Ambitious professionals building side projects after work.

Examples: Consultants, Product Managers, Engineers, Designers, Marketers, Operators.

---

## User Types

- **Dreamer** — Has ideas. Needs builders.
- **Builder** — Can build products. Needs opportunities.
- **Operator** — Can organize execution. Needs teams.
- **Creator** — Can market and design. Needs products.
- **Explorer** — Wants to build. Needs direction.

---

## Product Flywheel

Idea → Validation → Roadmap → Team Formation → Building → Launch → Discovery →
New Users → More Projects

---

## Core User Journey

**Day 1:** Join Forge, complete onboarding, receive AI roadmap, receive collaborator recommendations.

**Week 1:** Join project, create project, meet collaborators.

**Month 1:** Build MVP, complete milestones, launch something.

---

## MVP Philosophy

The MVP should focus on one thing: helping users move from idea to momentum.

Every feature should answer: *Does this help users build?* If not, it does not
belong in the MVP.
